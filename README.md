#  Deadlock timer in C **[deadlock_timer.c](deadlock_timer.c)**

## How to run
### Prerequisites
- Kate text editor
- Ubuntu Linux (24.04.1)

### Steps
1. Write the code in Kate text editor then save the file in the directory as deadlock_timer.c
2. Use the gcc (C compiler to run the code)
3. In Linux type: gcc deadlock_timer.c -o deadlock_timer -pthread
4. Then type to run ./ deadlock_timer  (specify the path where the file is located)
5. When code is being executed it creates **[activity_log.txt](activity_log.txt)**

### Output:
1. Terminal display status of process while code runs
2. Same information is written in activity log file in sequential entries
